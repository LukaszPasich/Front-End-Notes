JS Basics

Data types

JavaScript has 8 Datatypes:

  1. String
  2. Number
  3. BigInt
  4. Boolean
  5. Undefined (falsy)
  6. Null (falsy)
  7. Symbol
  8. Object
    • An object
    • An array
    • A date

You can use the JavaScript typeof operator to find the type of a JavaScript variable.
The typeof operator returns the type of a variable or an expression:

typeof "John" // Returns "string"
typeof 3 // Returns "number"

Comments

Single line comments

Single line comments start with // .
Any text between // and the end of the line will be ignored by JavaScript (will not be executed).

Single line comment before the code line:

// Change heading:
document.getElementById("myH").innerHTML = "My First Page";

Single line comment at the end of the line to explain the code:

let x = 5; // Declare x, give it the value of 5

Multi-line comments

Multi-line comments start with /* and end with */.
Any text between /* and */ will be ignored by JavaScript.

/*
The code below will change
the heading with id = "myH"
and the paragraph with id = "myP"
in my web page:
*/
document.getElementById("myH").innerHTML = "My First Page";
document.getElementById("myP").innerHTML = "My first paragraph.";

Docstrings

Docstrings are used to describe the JavaScript function. Docstrings are contained between /** and */ .
You can just type /** on the line right above the function and hit Return/Enter. This will bring up the generic docstring for that function, that you can then complete with detailed descriptions and specifications.

/**
* Description of what the function does
* @param {*} array
* @returns
*/
function foo(bar, optionalArg) {
  return 'baz';
}

Variables

Declaring variables

JavaScript variables can be declared in 4 ways:

  • Automatically
  • Using var
  • Using let
  • Using const

let

Variables defined with let have Block Scope.
Variables defined with let cannot be Redeclared.
Variables defined with let can be Reasigned.
Variables defined with let must be Declared before use.
Variables defined with let are not hoisted.
let does not bind to this.

const

Variables defined with const have Block Scope.
Variables defined with const cannot be Redeclared.
Variables defined with const cannot be Reasigned.
Variables defined with const must be assigned value when they are declared.
Variables defined with const are not hoisted.
const does not bind to this.

Always declare variable with const when you know that the value should not be changed.

Use const when you declare:

  • A new Array
  • A new Object
  • A new Function
  • A new RegExp

Template literals

Synonyms:

  • Template literals
  • Template strings
  • String templates
  • Back-tics syntax

Quotes Inside Strings

With template literals, you can use both single and double quotes inside a string:

let text = `He's often called "Johnny"`;

Multiline Strings

Template literals allows multiline string:

let text =
`The quick
brown fox
jumps over
the lazy dog`;

Variable Substitution

Template literals allows variables in strings:

let firstName = "John";
let lastName = "Doe";
let text = `Welcome ${firstName}, ${lastName}!`;

Expression Substitution

Template literals allows expressions in strings:

let price = 10;
let VAT = 0.25;
let total = `Total: ${(price * (1 + VAT)).toFixed(2)}`;

HTML Templates

let header = "Templates Literals";
let tags = ["template literals", "javascript", "es6"];

let html = `<h2>${header}</h2><ul>`;
for (const x of tags) {
    html += `<li>${x}</li>`;
}

html += `</ul>`;

Escape characters

The backslash (\) escape character turns special characters into string characters:

  • \b - Backspace
  • \f - Form feed
  • \n - New line
  • \r - Carriage return
  • \t - Horizontal tab
  • \v - Vertical tab
  • \' - Single quote
  • \" - Double quote
  • \\ - Backslash
